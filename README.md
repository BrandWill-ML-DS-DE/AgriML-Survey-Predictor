# AgriML-Survey-Predictor
A predictive modeling pipeline for agricultural adoption, featuring propensity score weighting and NLP-driven sentiment analysis. 

### Core Features
- **Propensity Weighting:** Corrects for non-response bias in statewide farmer surveys using Random Forests.
- **NLP Integration:** Extracts qualitative sentiment from farmer feedback to improve predictive accuracy.
- **MLP Architecture:** Utilizes Multi-Layer Perceptrons for adoption willingness classification.
